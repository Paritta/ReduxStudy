Main Technology Stack
===
* React.js
* React-Redux
* react-router-dom
* Redux
* Redux-saga
* Redux-actions
* Redux-form
* React-redux-firebase
* Styled-Component
* open-color
* Redux Devtools

Feature
===
* 인증
* 페이스북, 구글 인증
* 글 등록, 삭제
* 좋아요 
* 댓글
* 스터디 필터링 
* 이미지 업로드
* 인기 순 정렬

Installation
===
1. Dowload
```
$ git clone https://github.com/Paritta/ReduxStudy
```
2. Change folder
```
$ cd Ajax-Portfolio
```
2. Set up
```
$ yarn
```
3. execute
```
$ yarn start
```
